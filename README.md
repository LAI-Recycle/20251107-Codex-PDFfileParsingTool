# PDF 內容檢視與進階 OCR 工具

一個強大且基於純前端技術的 PDF 解析器。所有檔案解析與影像辨識（OCR）都在您的瀏覽器本地端進行，**絕對不會將敏敏感文件上傳至任何伺服器**，確保極致的資安與隱私。

## ✨ 核心特色與新功能

- **本地端安全解析**：完全依賴瀏覽器 CPU 處理，不依賴外部後端。
- **逐頁混合解析**：支援擷取 PDF 內建文字。當遇到掃描頁面時，自動退回使用 `Tesseract.js` 進行影像 OCR。
- **進階參數調校 (NEW!)**：
  - **Y 軸斷行靈敏度**：對於排版擁擠或字體大小不均的 PDF，可動態調整文字斷行規則。
  - **OCR 解析度倍率**：自由調整影像縮放倍率（2x ~ 6x），放大倍率能讓小字更清晰，提升辨識率。
  - **OCR 二值化門檻**：自由調整黑白轉換門檻（50 ~ 250）。可過濾雜訊髒污的背景，或搶救太淡的字元線條。
- **一鍵匯出 TXT (NEW!)**：解析完成後，可將全部頁面的結果打包，一鍵下載為文字檔，方便後續編輯。
- **記憶體防護 (NEW!)**：內建 `page.cleanup()` 機制，確保處理百頁以上文件時，舊分頁指標與 Worker 被安全釋放，防堵瀏覽器崩潰。

## 🚀 開始使用

1. 直接在瀏覽器裡雙擊打開 `index.html`（或拖曳至瀏覽器視窗）。
2. 在**設定區塊**調整您需要的參數（若不確定，可先使用預設值）：
   - Y 軸斷行靈敏度：4
   - OCR 解析度倍率：3x
   - OCR 二值化門檻：175
3. 點擊「選擇檔案」上傳 PDF，確認大小後按「開始上傳並解析」。
4. 等待狀態跑完每一頁。過程中您可看見結果區即時呈現解析進度。
5. 點擊右下方綠色的 **「匯出 TXT」** 按鈕來儲存結果。

> **提示**：第一次執行 OCR 會從 CDN 暫存 Tesseract 語言資料包，依網路速度可能需要幾秒鐘準備。一旦下載完畢，後續都會在電腦本地端運作。

## 🔌 離線部署 / 內網環境

如果需要在完全沒有網路的實體內網環境中使用：

1. 下載以下檔案並放置於專案資料夾中：
   - `pdf.min.js` 與 `pdf.worker.min.js`
   - `tesseract.min.js` 及其依賴的 WebAssembly 引擎
   - 繁體中文與英文語言資料包（`.traineddata`）
2. 修改 `index.html` 與 `app.js` 中的 CDN URL 為相對路徑（例如：`./vendor/pdf.min.js`）。
3. 依照 Tesseract.js 官方文件，在 `createWorker` 設定中透過 `corePath` / `langPath` 指定本地語言包的位置。

## ⚠️ 已知限制與優化空間

- 目前程式是以 `for` 迴圈循序漸進處理 OCR 畫布（考慮到記憶體回收的穩定度）。若有四核心以上的設備，未來可重構為 Scheduler + Worker Pool 架構來進行平行運算。
- 若掃描件帶有嚴重的旋轉（歪斜），Tesseract 可能會產出亂碼。極度歪斜的文件仍建議先透過外部影像軟體校正。
